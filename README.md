# Project-1
finding the l-th digit in the sequence of all integers concatenated from 1 up to n, and identifying which number the digit came from.

1. Create a single continuous string of digits from 1 up to n
2. Check if the requested position l is within the length of the string
3. The l-th digit is at index l-1 in the string
4. Find the number the target digit belongs to (The Source Number)
